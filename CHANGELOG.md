All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-03

### Added
- Initial release
- **One-click commit message generation** from Source Control panel
- **Three commit styles**:
  - Conventional (feat/fix/docs with scope)
  - Simple (one-line summary)
  - Detailed (full description with breaking changes)
- Configurable diff line limit for cost control
- Privacy-first design (no code stored)
- Powered by Claude AI (Anthropic)
