# Changelog

All notable changes to the "Commit AI" extension will be documented in this file.

## [1.0.0] - 2026-02-02

### Added
- Initial release
- Generate commit messages from staged changes
- Three commit styles: conventional, simple, detailed
- Source Control integration with sparkle button
- API key configuration via command or settings
- Configurable diff size limits for cost control
- Options to use, edit, or regenerate messages
