## How to Contribute

### Reporting Bugs

1. Check if the bug has already been reported in [Issues](https://github.com/CruceSaunders/commit-ai-vscode/issues)
2. If not, create a new issue using the Bug Report template
3. Include as much detail as possible

### Suggesting Features

1. Check existing [Issues](https://github.com/CruceSaunders/commit-ai-vscode/issues) for similar suggestions
2. Create a new issue using the Feature Request template
3. Describe your use case clearly

### Submitting Code

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/amazing-feature`)
3. Make your changes
4. Test your changes locally
5. Commit with a clear message (`git commit -m 'Add amazing feature'`)
6. Push to your fork (`git push origin feature/amazing-feature`)
7. Open a Pull Request

## Development Setup

1. Clone your fork
2. Run `npm install`
3. Press F5 in VS Code to launch the Extension Development Host
4. Test your changes in the new VS Code window
5. Make changes and reload to test

## Code Style

- Keep code clean and readable
- Use meaningful variable names
- Comment complex logic
- Follow VS Code extension guidelines
- Test before submitting
